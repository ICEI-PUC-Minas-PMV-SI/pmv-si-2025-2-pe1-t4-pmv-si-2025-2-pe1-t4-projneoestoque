# Testes de Software

**Caso de Teste** | **CT01 - @ obrigatorio no Login**
 :--------------: | ------------
**Procedimento**  | Inserir um usuario para logar no site que obrigatoriamente possua um @ 
**Resultado esperado** | Impedir caso não haja @
**Dados de entrada** | Inserção de dados válidos no formulário de cadastro
**Resultado obtido** | Sucesso

**Caso de Teste** | **CT02 - Impedir numeros inferiores a 0**
 :--------------: | ------------
**Procedimento**  | Inserir um numero invalido no sistema
**Resultado esperado** | Aviso do sistema que a operção é invalida
**Dados de entrada** | Inserção de dados válidos no formulário de cadastro
**Resultado obtido** | Sucesso

**Caso de Teste** | **CT03 - Filtros**
 :--------------: | ------------
**Procedimento**  | Buscar itens por filtros 
**Resultado esperado** | Achar o profuto esperado filtrando sua categoria
**Dados de entrada** | Inserção de dados válidos no formulário de cadastro
**Resultado obtido** | Sucesso

**Caso de Teste** | **CT04 - Teste de buscas**
 :--------------: | ------------
**Procedimento**  | Inserir o nome do produto, ou palavras chaves para localizalo no sistema
**Resultado esperado** | Encontrar o item
**Dados de entrada** | Inserção de dados válidos no formulário de cadastro
**Resultado obtido** | Sucesso
